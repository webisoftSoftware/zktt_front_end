# zKTT Front End

### Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation
Clone the repository:
   
   ```sh
   git clone https://github.com/webisoftSoftware/zktt_front_end.git
   cd zktt_front_end
   ```
   
Install dependencies:
   
   ```sh
   npm install
   ```

### Running the App
To start the  server:
    
   ```sh
   npm start
   ```

## Project Structure

- app
  - public
    - index.html
    - manifest.json
  - src
    - App.js
    - index.js
    - styles.css
    - util.js
